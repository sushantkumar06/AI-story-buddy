# Peblo Story Buddy

A Flutter application that narrates a story using Text-to-Speech and then presents an interactive quiz based on the story.

## Features

- Story narration using Flutter TTS
- Interactive quiz after story completion
- Riverpod state management
- Haptic feedback
- Smooth auto-scrolling
- Correct/Incorrect answer feedback
- Success celebration screen
- Confetti animation
- Responsive UI

## Tech Stack

- Flutter
- Dart
- Flutter Riverpod
- Flutter TTS
- Flutter Animate
- Confetti
- Vibration

## Project Structure

lib/
├── core/
├── features/
│ ├── story/
│ └── quiz/
└── main.dart

## How to Run

1. Clone the repository
2. Run:

flutter pub get

3. Start emulator/device
4. Run:

flutter run
